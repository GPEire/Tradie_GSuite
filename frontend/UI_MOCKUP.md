## Key UI Elements:

### 1. **Sidebar Panel** (Right side of Gmail)
- **Header**: "Projects" title with refresh and collapse buttons
- **Search Bar**: Search projects by name, address, or client
- **Project List**: 
  - Each project shows folder icon, name, status
  - Address with location icon
  - Client name with person icon
  - Email count and last activity time
  - Warning icon for projects needing review
- **Footer**: Project count and notification bell icon (ğŸ”” 3)

### 2. **Notification Toast** (Top right corner)
- Appears when new emails arrive or alerts occur
- Shows notification type icon (info, warning, error)
- Title and message
- Action button (e.g., "View Email")
- Auto-dismisses after 5 seconds (10s for errors)

### 3. **Notification Center** (Drawer)
- Opens from notification bell icon
- Shows all notifications with:
  - Category badges (New Email, Low Confidence, etc.)
  - Timestamp ("2 hours ago")
  - Action buttons
  - Mark as read / Delete options

### 4. **Project View Modal** (Overlay)
- Opens when clicking a project in sidebar
- Tabbed interface:
  - **Emails Tab**: Chronological list with previews
  - **Attachments Tab**: Grouped by file type
  - **Contacts Tab**: Client contact cards
- Settings button for project management

### 5. **Visual Design**
- **Colors**: Gmail blue (#1a73e8) for primary actions
- **Typography**: Google Sans font family
- **Icons**: Material-UI icons
- **Spacing**: Clean, modern layout with proper padding
- **Dark Mode**: Automatically adapts to system preference

## Color Scheme:
- **Primary Blue**: #1a73e8 (Gmail blue)
- **Success Green**: For completed/active states
- **Warning Orange**: For low confidence or review needed
- **Error Red**: For errors or critical alerts
- **Background**: White (light) / #202124 (dark)
- **Text**: #202124 (light) / #e8eaed (dark)
